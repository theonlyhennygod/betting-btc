## Features

*   Sports Betting (Upcoming & Live)
*   Fantasy Contests
*   Betting Pools
*   Lightning Network Payments (Deposit/Withdraw)
*   Taproot Escrow (Conceptual)
*   Live Chat
*   User Authentication via LN Wallet
*   Leaderboards & Tournaments
*   Referral System
